# 项目需求

## 项目题目

基于 EfficientFormerV2 与增强训练策略的 EuroSAT 遥感土地利用图像分类系统设计与实现。

## 核心目标

构建一个基于深度学习的遥感土地利用图像分类系统，在 EuroSAT 数据集上完成模型训练、评估、改进对比和可视化展示。

## 课程要求对应关系

- 选择不重复题目：本题使用 EfficientFormerV2 和 EuroSAT，避开已选题列表中的模型与数据集组合。
- 使用深度学习算法：以 EfficientFormerV2 为基础模型，使用迁移学习完成图像分类。
- 使用数据集：使用 EuroSAT 遥感图像数据集，可按实验条件使用全量数据或子集。
- 理解代码：代码需要保持清晰结构，并在关键位置写明变量、参数和函数作用。
- 改进算法：在基线模型基础上加入增强训练策略并做对比实验。
- 完成界面：后续使用 Streamlit 实现上传图像并输出分类结果的演示界面。
- 完成报告：根据结题报告模板整理研究背景、方法、实验、结果和总结。

## 主要交付物

- 可运行的 Python 项目代码。
- 数据加载、训练、评估和预测流程。
- 增强训练策略与基线对比结果。
- 训练曲线、混淆矩阵、预测示例等实验图表。
- 简单 Web 演示界面。
- 开发日志和项目文档。
- 结题报告写作素材。

## 非目标

- 不在 GitHub 上传课程原件、数据集、模型权重或本地输出。
- 第一阶段不追求完整训练结果。
- 不把 Web 演示界面作为学术创新点，只作为课程展示交付物。
